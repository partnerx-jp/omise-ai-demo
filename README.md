# omise-ai-demo

「お店AI」の体験デモ用フロントエンド。**掲載データはすべて架空**で、実在の店舗・個人・取引先とは一切関係ありません。

LINE公式アカウントのリッチメニューから開く LIFF 画面と、デモ用の表示テンプレートを GitHub Pages で配信します。

## 構成（単一ソース設計）

| パス | 役割 |
|---|---|
| `data/demo-data.json` | デモ全体の**唯一のデータソース**（架空のベーカリー3店舗）。LIFF も bot も同じこの1ファイルを読む |
| `flex/*.flex.json` | LINE Flex メッセージのテンプレート |
| `liff/*` | LIFF 画面（GitHub Pages で配信・Chart.js） |

## 方針

- 🔒 **認証情報（アクセストークン等）はこのリポジトリに一切含まれません**。鍵は連携先（n8n）の中だけに存在します。
- 🙈 各ページに `noindex` を付与し、検索エンジンにインデックスされないようにしています。
- ✏️ 数字や文面の変更は `data/demo-data.json` を1か所直すだけで、LIFF・bot 双方に反映されます。
